# YURTA.safe — AI-анализатор договоров аренды жилья

**YURTA.safe** — это юридическая AI-платформа, созданная для защиты студентов и молодежи Казахстана при найме жилья. Проект автоматизирует анализ юридических документов, подсвечивает критические риски и скрытые уловки в соответствии с законодательством РК (включая ГК РК), гарантируя полную конфиденциальность данных пользователей.

---

**YURTA.safe** позволяет проверить договор аренды за **10 секунд** без участия дорогостоящих юристов. Платформа переводит юридический язык на понятный человеческий и генерирует готовые правовые правки для арендодателей.

### Privacy-First Architecture (Защита персональных данных)
Безопасность пользователей — ключевой приоритет платформы:
1. **Локальная обезличенность:** На этапе загрузки файл проходит через локальный скрипт анонимизации.
2. **Анонимизация чувствительных данных:** Все личные данные (ФИО, ИИН, номера телефонов, реквизиты, адреса) автоматически находятся и заменяются на защищенные маркеры/заглушки.
3. **Безопасная передача:** Сырые и персональные данные **никогда** не отправляются на внешние серверы или сторонние API.

## Как это работает

1. **Загрузка документа:** Пользователь загружает PDF/PNG файл договора и указывает свою роль в сделке.
2. **Локальная очистка:** Скрипт маскирует ИИН, ФИО и другие данные на безопасные маркеры.
3. **ИИ-Аудит рисков:** Очищенный текст анализируется AI-моделью на соответствие Гражданскому кодексу РК.
4. **Формирование отчета:** Пользователь получает:
   * Оценку уровня риска и разбор «подводных камней».
   * Ссылки на конкретные статьи ГК РК.
   * Автоматически сгенерированный **Протокол разногласий** (готовый текст юридических правок для отправки арендодателю).
